# 🌿 KeenKeeper

**KeenKeeper** is a personal relationship management tool designed to help you stay connected with the people who matter most. Built as part of a CSE academic assignment, it tracks your interactions and reminds you when it's time to reach out to a friend.

---

## 🚀 Technologies Used

- **React.js** (Frontend Library)
- **Tailwind CSS** (Styling)
- **Vite** (Build Tool)
- **React Router** (Navigation)
- **Lucide React** (Icons)

---

## ✨ Key Features

* **Friend Interaction Tracking:** Keep a precise record of how many days have passed since you last contacted each friend, helping you maintain meaningful connections.
* **Visual Status Indicators:** Intuitive color-coded badges (**On-Track**, **Upcoming**, **Overdue**) that instantly show you who needs your attention at a glance.
* **Responsive Dashboard:** A clean, modern dashboard featuring summary statistics (Total Friends, Overdue, Interactions) and a beautifully organized grid of 12+ friend profiles.

---

## 🛠️ How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/keenkeeper.git](https://github.com/your-username/keenkeeper.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd keenkeeper
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```

## 📂 Project Structure

- `/src/assets`: Project images and icons.
